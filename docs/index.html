How It Works

Every line gets a unique fingerprint‚Äîa 2-character hash of its content. When your AI agent wants to edit line 2, it references 2:7f instead of just "line 2".

If someone inserts a line above, the old line 2 becomes line 3‚Äîbut its hash stays 7f. Hashfile finds it automatically.

Quick Start

// 1. Download the binary for your platform
curl -L https://github.com/mrorigo/hashfile-mcp/releases/download/v0.2.0/hashfile-mcp-YOUR_PLATFORM.tar.gz | tar xz

// 2. Add to your MCP client config (e.g., Claude Desktop)
{
  "mcpServers": {
    "hashfile": {
      "command": "/path/to/hashfile-mcp"
    }
  }
}

// 3. Use the tools in your AI agent
read_text_file({ "path": "/path/to/file.txt" })
edit_text_file({
  "path": "/path/to/file.txt",
  "file_hash": "abc123...",
  "operations": [...]
})
