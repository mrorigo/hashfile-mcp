Quick Start

// 1. Download the binary for your platform
curl -L https://github.com/mrorigo/hashfile-mcp/releases/download/v0.1.0/hashfile-mcp-YOUR_PLATFORM.tar.gz | tar xz

// 2. Add to your MCP client config (e.g., Claude Desktop)
{
  "mcpServers": {
    "hashfile": {
      "command": "/path/to/hashfile-mcp"
    }
  }
}

// 3. Use the tools in your AI agent
read_text_file({ "path": "/path/to/file.txt" })
edit_text_file({
  "path": "/path/to/file.txt",
  "file_hash": "abc123...",
  "operations": [...]
})
